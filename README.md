## 🛰️ Модули
- [ ] Пользователь (профили, анкеты и их настройки)
    - [ ] Репутация (выдача реакций на сообщения)
- [ ] Экономика (мешок, фарма, ириски)
- [ ] Модерация (бан, мут, кик, варны)
    - [ ] Апелляции (возможность написать заявку на разбан через бота) 
- [ ] Настройки сервера
