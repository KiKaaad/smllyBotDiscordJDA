# 🦧 Discord bot Juni#6593
Этот бот - аналог [Iris | Чат-менеджер](https://teletype.in/@iris_cm) для дискорда и все команды будут практически полностью индентичны

## Модули
- Пользователь (профили, анкеты и их настройки) | Завершен на 0%
  - Репутация (выдача реакций на сообщения) | Завершен на 0%
- Экономика (мешок, фарма, ириски) | Завершен на 0%
- Модерация (бан, мут, кик, варны) | Завершен на 0%
  - Апелляции (возможность написать заявку на разбан через бота) | Завершен на 0%
- Настройки сервера | Завершен на 0%
И многое другое позже..

## Команды
- [X] Пинг | Ping
- [ ] Анкета
  - [ ] Профиль
- [ ] Ферма
  - [ ] Мешок
  
## Для разработчиков  
Бот использует библиотеку JDA

Для работы в будущем создайте .env в главной директории своего проекта, добавьте Token=ВАШ_ТОКЕН_БОТА

Пожалуйста, если используете код моего бота, укажите авторство в описании, например: 
- Создан на базе smlly juni
- Автор большей части кода - t.me/kikaaad

Спасиба 🩷
